# Erste Versuche in R

Dieses Projekt enthält erste Analysen mit R zu Zeitreihen.

## Inhalt
- Datenaufbereitung
- Visualisierung
- ARIMA-Modelle und Forecasts
- Zusammenhangsanalyse (VAR / Granger)

## Dateien
- `analysis.R` – vollständiger Analysecode
- `data/` – verwendete CSV-Daten
- `plots/` – erzeugte Grafiken

## Pakete
- tidyverse
- forecast
- tseries
- vars

## Hinweis
Dieses Projekt dient Lern- und Analysezwecken.
